# Amazon Vinyl

Amazon Vinyl is an HTML5 playback engine for media streaming.

Current features supported:

- Dash streaming
- HLS streaming (fMP4 via MSE on all browsers, native on Safari)
- Progressive streaming
- ABR - Adaptive Bitrate Switching for Dash and HLS tracks
- Prefetching for Dash tracks
